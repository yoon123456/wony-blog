# Next.js SEO 체크리스트

애드센스 수익화를 목표로 한다면 검색 유입이 핵심입니다.

## 1. 메타데이터 일관성

- 페이지별 title, description 작성
- canonical URL 설정
- Open Graph, Twitter 카드 설정

## 2. 크롤링 접근성

- `robots.txt` 허용 정책 설정
- `sitemap.xml` 자동 생성

## 3. 구조화 데이터

- `BlogPosting` 스키마로 게시글 정보를 제공
- 발행일, 작성자, 본문 주제 정보를 명확하게 표현

## 4. 콘텐츠 품질

- 문제 해결 중심의 긴 글
- 목차/소제목으로 스캔 가능한 구조
- 내부 링크를 통한 체류 시간 증가

이 체크리스트를 기본으로 유지하면 색인 안정성과 검색 가시성이 크게 개선됩니다.
