# Next.js 마크다운 발행 워크플로우

콘텐츠 운영은 발행 빈도가 핵심입니다.
작업 순서를 단순화하면 생산성이 크게 올라갑니다.

## 발행 흐름

1. 키워드 리서치로 제목 후보 3개 작성
2. frontmatter(`title`, `description`, `date`, `category`) 먼저 확정
3. 본문 초안 작성 후 소제목 구조 정리
4. 내부 링크/카테고리 점검
5. 발행 후 Search Console 색인 요청

## 운영 팁

- 동일한 템플릿으로 글을 작성하면 속도가 빨라짐
- 매주 같은 요일에 1~2개 발행 루틴 고정

이 흐름만 유지해도 블로그가 꾸준히 성장합니다.
